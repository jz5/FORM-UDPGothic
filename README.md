# 帳票 UDPゴシック (FORM UDPGothic)

**モリサワ [BIZ UDPゴシック](https://github.com/googlefonts/morisawa-biz-ud-gothic) の「1」と「１（全角）」の幅だけを変更して、0〜9を等幅にしたフォントです。**

※ BIZ UDPゴシックは、0〜9のうち1だけ幅が異なる。
